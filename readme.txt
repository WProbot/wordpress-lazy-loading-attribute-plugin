A simple plugin to add the lazy loading attribute to images and iframes contained within post content.

== Description ==
This plugin adds the lazy loading attribute to images and iframes contained within post content for [supported browsers](https://caniuse.com/#feat=loading-lazy-attr).

**Requirements**

* WordPress 4.8 or higher
* PHP 5.4 or higher

**GDPR Safe**

This plugin _does not_ use or track _any user data_, either from visitors or users of WP Admin.

== Installation ==
1. Simply download to plugin ZIP, install and activate.

Once activated, it automatically adds the lazy loading attribute to images and iframes contained within post content.

There are no settings for this plugin.

== Frequently Asked Questions ==
**What happens in browsers that do not support the lazy loading attribute?**

The image/iframe will display normally, without lazy loading.
